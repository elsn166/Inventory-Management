Introduction

Admax is a responsive admin template powered by Twitter Bootstrap Framework. Admax-admin has a simple yet powerful set of UI elements and plugins to get your work done in the fastest time possible, with a user friendly and clean design to appeal to your customers.

We really care for our buyers and so in case if you have any question or feedback, please feel free to get back to us.

Structure

We have tried to follow the standards and modular structure while developing the theme. Following sections are explaining the theme File & Folder, structure, html file struture and js.

File & Folder Structure



├── Admax-Admin main folder
├── dist/css/
│   └── All css files.
├── assets/plugins/Bootstrap/
│   └── All Bootstrap files including css & js files
├── dist/images/

└── dist/js/

└── assets/css/
    └── All CSS files

HTML Structure

CSS

This template using the lot of css files, some of are from third parties. The main css files are bootstrap, style.css, font-awesome, these files are using into all html files and must be included to work it properly.

Files are explained below:

File Description
plugins.css Admax-Admin uses the bootstrap v4.0.1. The core bootstrap file is being used in all the pages.
main.min.css We are using the single css file for this template, this file is main css file of this template.

Javascript

Admax-Admin uses jQuery, Bootstrap JS framework(at its core) and some of the third-party plugins. There are may more third party plugin which you can use according to your needs. The css is already containing matching style for these plugins so you will not need to do anything around it.

Files are explained below:

File Description
plugins.min.js These files are used at core of the theme.
custom.js You can use this file for add any jquery scripts
